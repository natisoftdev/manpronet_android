Unable to establish a connection with the database!
The following problems may have occurred:
  • This is the first time the application is started and the connection string has not yet been set;
  • A wrong connection string has been entered;
  • You are allowed to use the multi connection string but the main one has not been chosen yet;
  • There is no stable internet connection.
The connection string must be entered under Set connection string in the Settings menu.

You can access the Settings menu in different ways depending on whether the phone has the Menu button:
  		• With physical key Menu (where present): when opening the app press the Menu key
  		• With Settings menu : when opening the app, press the three dots at the top right
    and go to Set connection string .

    Multi Connection String :
    If you are enabled to use multiple connection strings and still cannot log in, select the button at the top right () and select one of the items from the list.

    When you set the different strings under the heading Set connection string in the Settings section you need to separate them with a comma (,) and the first one will be considered as the main one.

    In case of problems you can contact our technicians who will provide you with rapid assistance.
